# CHECKLISTS DE VALIDACIÓN — ContentOps Toolkit

Este archivo contiene los puntos de control clave para cada fase de la producción editorial automatizada.

## 1. Validación de Estrategia (Backlog)
- [ ] ¿La keyword tiene volumen real (DataForSEO)?
- [ ] ¿La intención de búsqueda está confirmada?
- [ ] ¿Se ha comprobado que no existe ya una URL que cubra esta intención en `03_MASTER`?
- [ ] ¿El cluster y el ángulo editorial son coherentes?

## 2. Validación de Brief SEO
- [ ] ¿Incluye keyword principal y secundarias?
- [ ] ¿La estructura de encabezados (H2/H3) es lógica?
- [ ] ¿El enlazado interno propuesto viene de `Commerce_Catalog.xlsx`?
- [ ] ¿El archivo Markdown está en `examples/briefs/`?

## 3. Validación de Redacción (Draft)
- [ ] ¿El tono es natural y profesional (sin hype)?
- [ ] ¿Se ha respetado el brief inicial?
- [ ] ¿Los enlaces internos sugeridos están integrados de forma fluida?
- [ ] ¿No se incurre en keyword stuffing evidente?

## 4. Validación técnica de Publicación
- [ ] ¿El archivo HTML es semántico e incluye metaetiquetas (titles y descripciones)?
- [ ] ¿No hay estilos CSS inline ni JS innecesario?
- [ ] ¿Los enlaces internos apuntan a las URLs finales del catálogo?
